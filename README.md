# StarvationOsmolarity
This repository contains data and scripts for the paper titled - "Storage polymer polyhydroxybutyrate promotes recovery and limits phenotypic heterogeneity in regrowth from carbon starvation in Methylobacterium extorquens", preprint available at - https://doi.org/10.64898/2025.12.12.690574

"colony_trajectories_overall.csv" used in figures 3 and S3 is currently unavailable in this repository, as it is too large a file. To receive it, contact kjadams@uidaho.edu.
